## Day 79: Navigation
Old-school style navigation. Kinda like `vim`, in a way. 
```text
[1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
[q, w, e, r, t, y, u, i, o, p]
[a, s, d, f, g, h, j, k, l, ;]
[z, x, c, v, b, n, m, n, ,, .]
Use WASD to move up, right, down, left.
Use [space] to type a space.
Use : to output the current character.
```
